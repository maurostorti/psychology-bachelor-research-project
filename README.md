## Abstract

This cross-sectional study examined whether political conservatism is associated with higher subjective well-being (SWB) in a European population and whether distinct news consumer profiles moderate this relationship.

Data collected via an online questionnaire from 262 European adults were analyzed using Latent Profile Analysis (LPA). Profiles were derived from six key indicators: platform use frequency, active and passive social media engagement, news avoidance, confirmation bias, media outlet political bias, and diet polarization.

The analysis successfully replicated four media profiles established in previous literature: *Social Media News Consumers, News Minimalists, High-Engagement Omnivores,* and *News Traditionalists*.

Hypothesis testing yielded only anecdotal evidence in favor of an ideological “happiness gap” (H1). However, it provided overwhelming evidence against the moderation effect (H2), showing that media diets do not significantly alter the relationship between ideology and subjective well-being.

Given sampling constraints and an ideological skew, this paper serves as a pilot framework. Future research should build upon this methodology using more heterogeneous and better-powered samples.

## Project Structure & Workflow

To run the analysis:

1. Open the project in R using the `.Rproj` file
2. Go to the `00_config.R` file and set:
   - The **outlet dataset path**
   - The **response dataset path**
3. Run the scripts in **numerical order** (00 → 06)

Alternatively, you can run:
- `05_*.R` and `06_*.R` directly for final model and results generation
